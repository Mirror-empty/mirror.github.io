# 01.vs插件markdown使用快捷

实时预览：Ctrl + Shift + P 调出主命令框，输入 Markdown，应该会匹配到几项 Markdown相关命令，选择Markdown: Open Preview to the Side，就能调出实时预览框了。

|key	|command|
|---|---|
ctrl + b|	粗体
ctrl + i|	斜体
ctrl + shift + [	|下一级标题
ctrl + shift + ]	|上一级标题
ctrl + M	|不换行数学公式
Alt + S	|删除线

### 生成目录

安装后点开md文件， 然后快捷键CTRL(CMD)+SHIFT+P，输入Markdown All in One: Create Table of Contents回车即可，
